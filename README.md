# Zomato Funnel Analysis

## Overview

This project analyzes the ordering funnel of a food delivery platform similar to Zomato to identify user drop-off points and understand conversion across the ordering journey.

## Funnel Stages

App Open → Restaurant View → Add to Cart → Checkout → Order Completion

## Tools Used

* Python
* Pandas
* Google Colab
* Matplotlib

## Analysis

* Calculated the number of users at each funnel stage
* Measured conversion rates between stages
* Identified the highest drop-off point in the funnel
* Visualized funnel performance using Python

## Key Insight

The checkout stage showed the highest drop-off, suggesting possible friction during payment or final pricing.

## Suggested Improvements

* Budget-based restaurant recommendations
* Earlier pricing transparency
* Simplified checkout flow
* Faster payment options
